webpack系列-19webpack与浏览器缓存

发表于 2019-08-28 | 分类于 webpack系列

根据之前的配置，假设文件上传至服务器中，没有加hash，如果页面内容有更改，浏览器刷新的时候，请求的还是原先的文件，也就是浏览器的缓存，因为名字没有变。现在我们在上线的webpack配置中加上hash

这个时候打包出来的文件就有了hash值。只要文件内容不变，hash值就不变，内容变了才变。
